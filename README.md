# 🛒 Customer Behavior Analysis
**Tech Stack:** `Python` | `SQL Server` | `Power BI` | `Advanced Excel`

## 📌 Project Overview
Analyzed **10M+ rows** of consumer data to decode purchasing patterns and optimize database performance. This project covers the end-to-end data pipeline: from raw data cleaning in Python to interactive reporting in Power BI.

## 🚀 Key Impact
* **$11M Revenue:** Identified untapped growth opportunities via customer segmentation.
* **15+ Hours Saved:** Automated weekly reporting via Python scripts.
* **22% Accuracy Boost:** Optimized a **500GB database** using SQL Server.

## 📁 Dataset
* **Source:** Sales and Customer Interaction logs.
* **Size:** 10M+ Rows / 500GB SQL Environment.
* **Features:** Customer IDs, Transaction Dates, Product Categories, and Revenue.

## 🛠️ Tools Used
* **Python:** `Pandas`, `NumPy`, `SQLAlchemy` (Cleaning & ETL).
* **SQL Server:** Advanced queries, joins, and database optimization.
* **Power BI:** DAX, Data Modeling, and Interactive Dashboards.
* **Excel:** Power Query and Final Reporting.

## 📈 Project Steps
1. **Data Loading:** Ingested raw datasets using Python.
2. **EDA & Cleaning:** Handled missing values and outliers in `Pandas`.
3. **Database Migration:** Migrated cleaned data to **SQL Server** for structured querying.
4. **SQL Analysis:** Ran complex queries to extract churn and retention metrics.
5. **Visualization:** Built a Power BI dashboard to track KPIs.
6. **Reporting:** Generated a final executive summary with actionable insights.
